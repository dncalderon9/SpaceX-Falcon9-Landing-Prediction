# SpaceX Falcon 9 Landing Prediction

## Project Overview

This project analyzes historical Falcon 9 launches to predict whether the first stage booster will land successfully.

## Objectives

- Collect launch data
- Perform data wrangling
- Conduct exploratory data analysis
- Build interactive dashboards
- Develop machine learning models

## Technologies

- Python
- Pandas
- SQL
- Plotly Dash
- Folium
- Scikit-Learn
- BeautifulSoup
- SpaceX REST API

## Results

- Dashboard
- Interactive Maps
- Machine Learning Accuracy ≈ 83%
